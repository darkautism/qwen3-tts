⚙️ Worker Configuration

Setup Instructions

Each worker runs on an ARM SBC (e.g. RK3588). Start workers on each machine:

Machine 1 — Talker
qwen3-tts worker --role talker --bind 0.0.0.0:9090 --big-cores

Machine 2 — Predictor (use --quant q4 for speed)
qwen3-tts worker --role predictor --bind 0.0.0.0:9091 --big-cores --quant q4

Machine 2 — Vocoder
qwen3-tts worker --role vocoder --bind 0.0.0.0:9092 --big-cores

Enter the host:port for each worker below. Changes are saved to the config file and reconnected immediately.
